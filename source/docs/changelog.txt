changes for version 1.4
- new feature:
  - add main menu
  - configuration is now saved when running the application.
  - add settings:
    - You can now automatically check for updates when you start the software.
    - The application starts with window.
    - Now you can turn on and off the feature to automatically receive links from the clipboard.
    - You can use control+tab to quickly move between tabs, control+shift+tab to go back to the previous tab.
    - add backup button: You can save your configuration and take it anywhere to use.
    - add restore button: you can restore the entire configuration if the application has very serious problems.
    - Skip silence (Recommended only for music): Silence will be ignored when moving to the next video or returning to the previous video.
    - Speak player status notifications: When this is selected all notifications in the player will be read clearly from the screen reader. If you turn it off, it will be completely silent, making you feel more comfortable when watching videos.
    - After Video Finishes: allows you to choose actions to be performed in the music player including:
        - Close the player: The music player will close after the video ends.
        - Replay video: you can allow the video to playback multiple times.
        - Do nothing: You can roughly understand that the player will do nothing if this action is selected.
    - Fast Forward Interval (Seconds): You can customize the fast forward time while watching a video.
    - Rewind Interval (Seconds): You can adjust the rewind time here.
  - Added a time jump system so you can jump to any place in the video you want.
  - Added start and end point selection in a video to split into many different music files.
  - Added advanced filtering when searching results.
  - smart playlist:
    - When you press enter on a result in the form of a playlist, a dialog box will appear to download the videos in that list.
    - If you press enter on each of those videos, the player will be launched to start watching the video.
    - When you use the keys to skip videos or go back, it will still follow the videos as a playlist in the playlist loading box.
    - To close the playlist box, please press the close button or use the ESC key.
- New shortcut:
  - control+shift+s: activate settings
  - control+g: go to time (only valid in player interface).
  - mở ngoặc vuông [: Start selection.
  - đóng ngoặc vuông ]: End selection.
  - alt+o: go to option menu (Use only in music player).
  - control+s: Save selection as.
  - control+shift+d: activate Video Description...
- fix bug:
  - The entire software has been optimized to make your experience more comfortable.
  - Fixed startup with window error when sound packs were not started correctly.
  - Fixed a bug related to the return to main window button.
  - Redesigned the context menu interface. it will be passed into the music player to return the enter key for the play action.
  - Fixed a bug related to buffer leaks.
  - The commenting system has been significantly upgraded. Now you can only copy a single comment instead of all of them like before.
  - Improved overall latency for overall software acceleration.
  - readme has been updated.
  - Updated libraries to hide unnecessary windows.
  - The application will run under administrator rights when you call it from anywhere on the computer.
  - Fixed the error of not being able to enter text in the input boxes. Please note that to be able to type text when running under administrator rights, please set the Vietnamese percussion instruments to run under administrator rights to be effective.
  - Fixed an issue where playlists could not be loaded in video viewing history.
  - Now all actions in the filter are working properly.
  - Fixed all errors with control+tab and control+shift+tab keyboard shortcuts.
  - Added property page to settings so you know where you are.
  - changed the search interface. Now the combo box will appear before the search button.

changes for version 1.3
- new feature:
  - Added a 2-second delay for better accessibility and implemented automatic loading of more results as you scroll down.
  - completely removed the notification as ToolTip.
  - add anti-scan system from windows security.
  - Now in the media player all notifications will be read through NVDA or sapi5 if NVDA fails.
  - add comments system to review comments from video.
- New shortcut:
  - control+t: Read Elapsed Time.
  - Ctrl + Shift + T: Read Total Duration.
  - 1 through 9: Jump to 10% through 90% of the video duration.
  - f2: view changelog.
  - R: Toggle Repeat.
  - Shift + N: Next Video.
  - Shift + B: Previous Video.
  - N: Toggle Auto Next (Recommended Videos).
  - Ctrl + W: Exit Application.
  - S / D / F: Change Speed (Fast / Normal / Slow).
  - (Minus) / = (Equal): Decrease / Increase the seek step (seconds jumped by arrows).
  - Ctrl + Shift + E to toggle Cinema Mode (Fullscreen).
  - ctrl+shift+c: activate comments video.
  - ctrl+enter: play as audio.
- new library:
  - nvdaControllerClient64.dll: for computers using 64bit architecture.
  - nvdaControllerClient32.dll: for computers using 32bit architecture.
- fix bug:
  - Removed annoying sound when using keyboard shortcuts in media player.
  - improved navigation when interacting with screen readers in lists.
- Fixed a critical error for full-screen systems when watching videos.
- Conflicting components have been removed.
- Improved Home/End key responsiveness: In the music player, the Home key now restarts the song immediately. In lists (search, history, favorites), the Home and End keys will help you jump to the top or bottom of the list quickly.
- Reduce waiting time when transferring songs: I have optimized the yt-dlp commands so that getting the next song URL is faster, helping to shorten the transfer time.
- Prevent the Escape key from closing applications: I have set the Escape key to be automatically ignored in the main window to prevent accidental closing of applications.
- Improved Home/End in the player: The Home key restarts the song immediately. The End key will jump to the end of the song to activate automatic song switching (if AutoPlay is enabled), making the experience smoother.
- Fix Auto Load More error: Now when you press the End key in the search list, the application will jump to the bottom and automatically load new results immediately.
- Reduce waiting time: The welcome screen (Splash Screen) has been reduced to 3 seconds so you can enter the application faster. Video loading commands have also been optimized to reduce latency when switching tracks.
- Fetching URL speed: Has been significantly improved by removing unnecessary data and setting more stringent connection timeouts. You'll see videos load faster when you skip tracks or go back.
- Automatically download results more smoothly: The application will now automatically download more results when you are 10 items away from the end of the list (instead of 5 as before), helping to prevent list scrolling from stalling.
- Increased overall performance: I've reduced latency in internal data processing loops (from 10ms to 1ms), making the app more responsive and "responsive" to your actions.
- Improved accessibility: Combo boxes have been redesigned with clear labels ("Select format (Tab to move)") and hints for screen readers, ensuring NVDA/JAWS users can use them easily and accurately.
- changed the way it works when rewinding to the end of the video using the end key.
- Updated the labels to be more screen reader friendly and the messages are clear so users can understand.
- Reduced wait time between search and play dialogs to speed up the user experience.
- Rewrote the entire yt-dlp library update system to ensure bugs are specifically addressed.
- Fixed the software exit bug so that included tools including **Description.exe** and **comments.exe** exit completely to avoid the situation of not being able to replace new files.
- Now when you update the yt-dlp library, the software will require restart to apply the latest update.
